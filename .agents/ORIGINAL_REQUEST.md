# Original User Request

## Initial Request — 2026-06-16T22:55:22Z

Build a web-based "Crisis-to-Action Translator" called **ENVIS** for a hackathon. The application allows users to upload stressful documents (like legal notices, bills, or eviction warnings) and uses AI to translate them into plain language, provide an interactive checklist of next steps, and generate a draft response.

## Requirements

### R1. Tech Stack & Framework
- Framework: Next.js (latest version)
- Styling: Tailwind CSS
- Animation: GSAP (GreenSock Animation Platform) for smooth, anxiety-reducing micro-animations and transitions.
- OCR Library: Tesseract.js (or similar client-side library) for parsing text from uploaded images/PDFs.
- AI Integration: Real API connection to an LLM (e.g., Gemini or OpenAI) using an API key from environment variables.

### R2. Brand Guide & Color Tokens
Implement the exact design system outlined in `brand.md`:
- Colors:
  - `--calm-sage`: `#7FA99B` (Primary brand/trust)
  - `--deep-pine`: `#2E4F4A` (Headings, primary text on light)
  - `--warm-sand`: `#EAE3D6` (Cards/surfaces)
  - `--soft-clay`: `#D9A283` (Gentle accent highlights)
  - `--sky-mist`: `#A9C5CE` (Secondary actions/info)
  - `--paper`: `#FAF7F2` (App background)
  - `--ink`: `#33403D` (Body text)
  - `--mist`: `#C9D2CE` (Borders/dividers)
  - `--attention`: `#E0B25C` (Warm, time-sensitive alert—NO red warnings)
- Dark Mode ("Night Calm"):
  - `--paper`: `#161E1C`
  - `--surface`: `#1E2826`
  - `--ink`: `#E4E9E6`
  - `--calm-sage`: `#8FBBAC`
  - `--mist`: `#2C3835`
- Typography: Use Google Fonts for `Fraunces` (headings) and `Inter` (body). Generous font size (17–18px body, line-height 1.6+).
- Texture: Include a subtle, light noise/grain overlay (2-4% opacity) to add warmth.

### R3. Document Ingestion
- Upload portal supporting image/PDF drag-and-drop.
- Real-time OCR text extraction with a visual GSAP scanning progress animation.
- A manual copy-paste text box.
- A library of preset stressful documents (e.g., "Hospital Bill", "School Suspension Notice") to allow instant testing.

### R4. The Calm Translation Dashboard & Features
- **Empathic Summary Panel**: A friendly 2-3 sentence overview that answers: "What is happening?", "Do I need to panic?", and "What is the main thing I need to do?".
- **Timeline/Deadlines**: Urgency dates color-coded with `--attention` or `--soft-clay`.
- **Smart Jargon Decoder**: Highlight legal/technical jargon words in the translated text. Hovering or clicking a highlighted word opens a popover defining the term in simple language.
- **Step-by-Step Action Checklist**: Dynamic, actionable, step-by-step list generated by the AI. Checklist items can be marked complete, updating a circular progress indicator.
- **Response Draft Assistant**: Auto-generated replies or appeal letters. Provide tone options (e.g., "Polite request", "Formal dispute"). Easy one-click copy-to-clipboard button.
- **Emergency Resource Directory**: Direct help contacts matching the document category (e.g., free legal clinics, food banks).

### R5. Visual & Motion Experience
- While the AI is processing, show a soft, breathing, solid-color disc (`--calm-sage`) gently scaling and fading opacity on a ~4-second breathing cycle.
- Streaming text effect as the plain-language summary is rendered.
- List items and cards should assemble one-by-one with a soft, springy GSAP settle animation.
- Button press states (scale down slightly to `0.98` on click) and card hover states (hover lifts cards 2-4px with deepening shadow).
